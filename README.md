# Aion Virtual Machine

Aion Virtual Machine (AVM) is a Blockchain VM based on Java virtual machine. It executes DApps written in Java, Scala, and Kotlin.


## Prerequisites

```
JDK 10 or higher
Apache Ant 1.10 or higher
```


## Build

```
ant
```


## Run the tests
```
ant test
```

## Build and test
```
ant && ant test
```
